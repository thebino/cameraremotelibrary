    ObjectFormat¶

    [Camera remote library]\ object ObjectFormat

    Object format is a 16-bit unsigned integer that represents the format of an image or data object that resides on a DSLR/DSLM.

    Setting bit 15 to 1 in the ObjectFormatCode indicates a vendor-defined format.

    All ObjectFormatCodes that represent image formats shall also have bit 11 set to 1, including those that are vendor-defined.
